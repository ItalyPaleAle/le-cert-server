```yaml
dnsProvider: "route53"
dnsCredentials:
  AWS_ACCESS_KEY_ID: "your-access-key"
  AWS_SECRET_ACCESS_KEY: "your-secret-key"
  AWS_REGION: "us-east-1"
```

See the [Lego Route53 documentation](https://go-acme.github.io/lego/dns/route53/) for the full list of supported environment variables.
